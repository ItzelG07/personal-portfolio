RUBIGINOSA

UX RESEARCH | UX DESIGN | KEY TAKEAWAYS

OVERVIEW: Hired to create an appointment scheduling app for a nail technician that is moving to a new salon.

ROLE:UX Researcher, UX/UI designer

TOOLKIT:Figma, Pencil and paper

UX RESEARCH

BACKGROUND

PROBLEM:The struggles of miscommunication with clients made me create this app to consider accessibility issues from the view of a nail technician.

GOAL: Users can access service information to determine their compatibility with the nail technician or quickly schedule an appointment for regular users.

01 Understanding the User

I conducted user research through primary and secondary research. I created surveys to find customer issues at nail salons that I could find solutions for and implement in my app. I researched direct and indirect competitors to find pain points and accessible elements about the website. I created user personas and problem statements to look into what a typical user may face when navigating through my app.

02 Refining the Design

Change colors for buttons based on importance and contrast.
Create featured that would improve the user flow such as grouping elements together so the users can have different options when interacting with the app.
Have universal icon that represent descriptions so that it can be understood by all audiences.

03 Responsive Design

The sitemap for Rubiginosa includes every screen the app can hold. I highlighted one user flow that I designed for. My user flow includes selecting the “No Polish” manicure service. The user can then schedule the appointment for March 7, 2024 and purchase with a credit card.
The Mobile and Tablet screen both have the same format for the homepage but the tablet shows more content below the fold. For the computer screen I created a two tile splash page where there is a image on the left and de3scription with CTA button on the right. Having the wto tile method covers most of the screen without loosing picture quality or causing stretched images.

04 Going Forward

When creating my app Rubiginosa, I wanted to make an impact with my accessibility consideration. Colors that were chosen for graphics, buttons, and typography is all intentional with keeping visual accessibility designs in mind. Graphic were kept simple to not be distracting from the content. Typography for buttons and descriptions is a sans serif font.
Create more user flows that explore the different sections in the Hamburger Menu.
Create translation options for the most spoken languages in the area.
Create a section for customer review. “Look at Others Wearing it” would show pictures linked to the clients instagram.
